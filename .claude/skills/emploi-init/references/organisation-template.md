**Les sous-dossiers doivent remplir le modele suivant:**

- Recherches-[cible].md
- CV_{{NOM_PRENOM}}-[cible].md
- LM_{{NOM_PRENOM}}-[cible].md
- Recherches-[cible]-[nom/service].md
- LinkedIn-[cible]-[prenom]_[nom].md
- Email-[cible]-[nom/service].md
- Pitch-[cible].md
- Entretien-[cible].md
- Suivi-[cible].md
  - [ ] CV Envoyer
  - [ ] LM Envoyer
  - [ ] Contact Envoyer
  - [ ] Recherche terminer
  - [ ] Entretien preparer
- Dossier "print" contenant les versions docx/pdf a la demande

## Process de recherche

1. Definir la cible
2. Rechercher les informations necessaires
3. Analyser les informations trouvees
4. Ecrire un CV personnalise
5. Ecrire une lettre de motivation personnalisee
6. Preparer un pitch personnalise
7. Preparer un entretien personnalise (question / reponse)
8. Faire des recherches sur les contacts potentiels
9. Preparer les messages personnalises au cas ou contacts sur LinkedIn
10. Initier un document de suivi "Suivi-[cible].md"
11. Creer un dossier "documents/[cible]" pour stocker tous les documents generes
